## 🔹 Flash Manager

### Smart File Transfer

**Just transfer your data from computer to computer**

---

### What is it for

Standard file explorer makes you do extra clicks. Flash Manager remembers your favorite folders and copies in one click.

Useful when moving to a new computer or working with a flash drive often.

---

### Who is it for

👥 **Moving to a new PC** — collect files on the old one, transfer to a flash drive, sort into folders on the new one.

💻 **Creative people** — test programs on different computers, transfer projects between workstations.

🎓 **Students** — home, library, university. The flash drive is always with you.

---

### Why you should try it

1. **PC to PC transfer** — no clouds, no registrations.
2. **Saved paths** — add a folder once, it's always there.
3. **Search across the entire flash drive** — find and copy.
4. **Drive analytics** — pie chart of largest files.
5. **Convenient ZIP and RAR extraction** — right in the program.
6. **Portability** — put it on a flash drive and run on any PC.

---

### How to run

**.exe file (ready program)**
Download and open. Nothing additional needs to be installed.

**.py file (source code)**
Requires Python to be installed. Download from the official website: python.org
After installation, run the file with the command `python flash_manager.py`

## 🔹 Flash Manager

### Умный перенос файлов

**Просто переносите данные с компьютера на компьютер**

---

### Для чего это

Стандартный проводник заставляет делать лишние клики. Flash Manager запоминает нужные папки и копирует в один клик.

Удобно, когда переезжаете на новый компьютер или часто работаете с флешкой.

---

### Для кого

👥 **Переезд на новый ПК** — собрали файлы на старом, перенесли на флешку, на новом разложили по папкам.

💻 **Креативным людям** — тестируете программы на разных компьютерах, переносите проекты между рабочими станциями.

🎓 **Студентам** — дома, в библиотеке, в универе. Флешка всегда с вами.

---

### Почему стоит попробовать

1. **Перенос ПК → ПК** — без облаков и регистраций.
2. **Сохранённые пути** — добавил папку один раз, она всегда под рукой.
3. **Поиск по всей флешке** — нашёл и скопировал.
4. **Аналитика накопителя** — диаграмма самых больших файлов.
5. **Удобная распаковка ZIP и RAR** — прямо в программе.
6. **Портативность** — положил на флешку и запустил на любом ПК.

---

### Как запустить

**Файл .exe (готовая программа)**
Скачайте и откройте. Ничего дополнительно устанавливать не требуется.

**Файл .py (исходный код)**
Требуется установленный Python. Скачайте с официального сайта: python.org
После установки запустите файл командой `python flash_manager.py`
